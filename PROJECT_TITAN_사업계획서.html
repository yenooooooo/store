📌 EXECUTIVE SUMMARY

이 문서는 생성형 AI 바이브코딩 방식으로 2개의 수익형 웹사이트를 동시 개발·런칭하여, 최단 기간 내 현금 흐름을 만들고 2억 원의 부채를 상환하기 위한 전략적 사업 계획서입니다.

🎯 핵심 전략 요약
  • Product 1: 셀러부스트 AI — 스마트스토어 판매자를 위한 AI 통합 상품 최적화 플랫폼
  • Product 2: 바이럴랩 AI — 유튜버/크리에이터를 위한 AI 바이럴 콘텐츠 엔진
  • 개발 방식: 바이브코딩 (Claude CLI + Next.js + Supabase + Vercel)
  • 수익 모델: SaaS 월정액 구독 + 종량제 크레딧 혼합
  • 런칭 타겟: 14일 내 베타 런칭, 30일 내 유료 전환 100명 목표
런칭 순서 전략: 셀러부스트 먼저, 바이럴랩은 D+15부터
📌 동시 개발이 아닌 순차 런칭 — 이유 3가지
  • ① 판매자는 돈과 직결: 셀러는 상세페이지가 매출에 즉시 영향 → 결제 전환이 빠름. 크리에이터는 효과 체감까지 시간이 걸림
  • ② 해자가 셀러부스트에서 더 강력: URL 크롤링 + 스마트스토어 HTML 내보내기는 ChatGPT가 절대 못 하는 기능. 바이럴랩은 상대적으로 ChatGPT와 차이가 적음
  • ③ 코드 70% 재사용: 인증·결제·UI·Claude API 모듈을 셀러부스트에서 먼저 만들면, 바이럴랩은 1주일 안에 추가 가능
왜 이 두 제품인가?
📦 셀러부스트 AI
  • 시장 규모: 스마트스토어 판매자 700만+
  • 월 신상품 등록 수: 수천만 건
  • 기존 솔루션: 없거나 UI가 구식
  • 단가: 건당 500~1000원, 체감 저렴
  • 재구매율: 매우 높음 (월정액 적합)
  • 바이럴 경로: 셀러 카페·블라인드
🎬 바이럴랩 AI
  • 시장 규모: 유튜버/크리에이터 200만+
  • 월 콘텐츠 필요량: 4~12개/인
  • 기존 솔루션: 영어 중심, 한국어 최적화 없음
  • 단가: 월 14,900원 (체감 ROI 매우 높음)
  • 재구매율: 매우 높음 (크리에이터 생존 도구)
  • 바이럴 경로: 유튜버 카페·디시·틱톡
🆚 "그냥 ChatGPT 쓰면 되는 거 아냐?" — 이 질문에 대한 답

솔직히 말하면, 단순 카피 생성만 원한다면 ChatGPT로도 된다. 하지만 우리 제품을 써야만 하는 이유는 명확하다.

😅 ChatGPT로 하면
  • 상품 정보를 매번 직접 복붙해야 함
  • 네이버 SEO 알고리즘을 ChatGPT는 모름
  • 생성 결과를 어디에도 저장 못함
  • A/B 테스트? 직접 여러 번 물어봐야
  • 스마트스토어 형식으로 내보내기 불가
  • 생성 기록·히스토리 관리 불가
  • 프롬프트를 직접 잘 짜야 결과가 좋음
✅ 우리 서비스는
  • URL 하나로 상품 데이터 자동 크롤링
  • 네이버 쇼핑 검색 알고리즘 맞춤 키워드 자동 삽입
  • 생성→편집→A/B→성과추적 한 곳에서
  • 클릭 한 번으로 3가지 변형 버전 즉시 생성
  • 스마트스토어 HTML 형식으로 바로 붙여넣기
  • 히스토리·버전 관리·팀 공유 모두 가능
  • 판매자 전용 프롬프트 최적화로 항상 좋은 결과
우리만의 4가지 핵심 해자 (Moat)
🔒 해자 #1 — 데이터 연동 (ChatGPT가 절대 못 하는 것)
  • 스마트스토어 / 쿠팡 URL 입력 → 상품 데이터 자동 크롤링 후 즉시 생성
  • 실제 구매 리뷰 데이터 분석 → "고객이 실제로 반응한 포인트" 자동 추출
  • 네이버 쇼핑 자동완성 검색어 API 연동 → 실시간 인기 키워드 자동 삽입
  • 경쟁사 상위 상품 구조 역분석 → 우리 상품에 자동 적용
🔒 해자 #2 — 워크플로우 전문화 (생산성 10배)
  • 생성 → 편집 → A/B 테스트 → 성과 추적 — 탭 이동 없이 한 화면에서
  • ChatGPT는 매번 컨텍스트를 다시 입력해야 하지만, 우리는 상품 정보 한 번 저장으로 끝
  • 히스토리 관리, 버전 비교, 팀원 공유 기능 내장
  • 생성된 카피 → 인스타그램/블로그/광고 문구 원클릭 변환
🔒 해자 #3 — 플랫폼 최적화 (한국 판매자 전용)
  • 네이버 쇼핑 SEO 알고리즘 맞춤 키워드 자동 배치 (ChatGPT는 이 데이터가 없음)
  • 스마트스토어 / 쿠팡 / 11번가 각각 다른 형식으로 자동 분리 출력
  • 플랫폼별 글자수 제한, 금지어, 필수 항목 자동 검증
  • 한국어 쇼핑 트렌드 맞춤 표현 자동 적용 (일반 AI는 영어 중심)
🔒 해자 #4 — 즉시 내보내기 (Copy & Done)
  • 스마트스토어 에디터 호환 HTML 형식으로 원클릭 복붙 — 재편집 불필요
  • 이미지 슬롯이 자동 배치된 레이아웃 템플릿 생성 (이미지만 교체하면 완성)
  • 카테고리별 검증된 레이아웃 구조 자동 적용 (의류/식품/전자기기 등)
  • 결과물 PDF 저장 / 팀원 공유 링크 생성 원클릭

📦 PRODUCT 1: 셀러부스트 AI

스마트스토어 판매자 AI 통합 상품 최적화 플랫폼

단순한 '글쓰기 도구'가 아닌 — 경쟁 분석 + SEO 키워드 자동 삽입 + 전환율 예측까지 제공하는 판매자 성장 플랫폼

1-1. 핵심 차별점 (타사에 없는 것들)
🔑 타사 대비 핵심 경쟁 우위 — 이것이 없으면 망한다
  • ① 경쟁상품 자동 분석 엔진: URL 입력 → 경쟁사 상위 상품 키워드·구조 역분석
  • ② 전환율 예측 스코어: AI가 생성한 상세페이지에 '예상 전환율 85점' 같은 점수 부여
  • ③ 네이버 SEO 자동 삽입: 네이버 쇼핑 검색 알고리즘 맞춤 키워드 자동 배치
  • ④ A/B 변형 3종 자동 생성: 하나 입력에 스타일이 다른 3개 버전 즉시 생성
  • ⑤ 상품 카테고리별 최적화 템플릿: 의류/식품/전자기기 등 카테고리별 맞춤 구조
  • ⑥ 이미지 분석 → 텍스트 연동: 상품 이미지 업로드 시 이미지 분석 후 설명 자동 생성
1-2. 핵심 기능 명세
Core Feature #1: 원클릭 상세페이지 생성기
  • 상품명, 카테고리, 핵심 특징 3가지 입력
  • AI가 제목(SEO최적화) + 서브타이틀 + 특징 5가지 + 사용 시나리오 + CTA 자동 생성
  • 네이버/쿠팡/11번가 포맷 각각 최적화 버전으로 분리 출력
  • 원클릭 복사 / 스마트스토어 에디터 직접 붙여넣기 최적화
Core Feature #2: 경쟁사 역분석 리포트
  • 네이버 쇼핑 키워드 입력 → 상위 10개 경쟁 상품 구조 자동 분석
  • '경쟁사가 쓰는 키워드 TOP 20' 리포트 생성
  • 내 상세페이지와 경쟁사 비교 갭 분석 차트 제공
Core Feature #3: 전환율 예측 AI 스코어보드
  • 생성된 상세페이지에 0~100점 '전환율 예측 점수' 부여
  • 점수 향상을 위한 구체적 개선 제안 3가지 자동 생성
  • '이 문장을 이렇게 바꾸면 +5점' 형태의 인라인 코칭
Core Feature #4: 마케팅 카피 원스톱 생성
  • 상세페이지 → 인스타그램 홍보 문구 자동 변환
  • 상세페이지 → 네이버 블로그 체험단 후기 초안 자동 생성
  • 상세페이지 → 네이버 파워링크 광고 문구 3종 자동 생성

🎬 PRODUCT 2: 바이럴랩 AI

유튜브·쇼츠·릴스 바이럴 콘텐츠 AI 엔진

단순 '스크립트 생성기'가 아닌 — 바이럴의 공식을 수치화하고, 후킹 강도를 점수로 보여주며, 채널 성장 전략까지 제시하는 크리에이터 성장 플랫폼

2-1. 핵심 차별점 (국내 유일 기능)
🔑 타사 대비 핵심 경쟁 우위 — 국내 유일 기능
  • ① 바이럴 공식 분석 엔진: 100만뷰+ 영상 패턴을 학습한 후킹 공식 자동 적용
  • ② 후킹 강도 점수 (Hook Score): 영상 첫 3초~15초 대본에 0~100점 점수화
  • ③ 조회수 예측 AI: 제목+썸네일 문구 입력 시 '예상 클릭률 범위' 제공
  • ④ 플랫폼별 완전 분리 최적화: 유튜브 10분 / 쇼츠 60초 / 릴스 30초 각각 다른 구조
  • ⑤ 채널 톤앤매너 학습: 내 기존 영상 3개 입력 → 내 스타일 그대로 새 스크립트 생성
  • ⑥ SEO 제목 A/B 10종 생성: 클릭 유도 심리 기법별로 10개 제목 후보 자동 생성
2-2. 핵심 기능 명세
Core Feature #1: 바이럴 스크립트 생성기
  • 주제 + 타겟 시청자 + 영상 길이 입력
  • 훅(Hook) → 공감대 형성 → 핵심 내용 → 반전/클라이맥스 → CTA 구조로 자동 분배
  • 각 섹션마다 '왜 이렇게 쓰나' 코칭 코멘트 삽입
  • 자막용 타임스탬프 초안 자동 생성
Core Feature #2: Hook Score 분석기
  • 작성한 대본의 첫 15초 분량 붙여넣기 → 즉시 점수화
  • '이 문장에서 시청자가 이탈할 가능성이 높다' 구체적 지적
  • 개선된 버전 3가지 즉시 제안
  • 100만뷰 영상과 내 대본 비교 레이더 차트 시각화
Core Feature #3: 제목·썸네일 문구 생성기
  • 주제 입력 → 클릭 심리 기법별 제목 10종 생성 (궁금증 유발 / 숫자 / 공포 / 공감 등)
  • 각 제목의 '예상 CTR 등급' 표시 (S/A/B/C 등급)
  • 썸네일에 들어갈 임팩트 문구 5종 자동 생성
Core Feature #4: 채널 성장 전략 리포트
  • 채널 주제 + 현재 구독자 수 입력
  • 알고리즘 최적화를 위한 주간 업로드 전략 자동 생성
  • 경쟁 채널 분석 기반 '공략 틈새 키워드' 20개 제안

크롤링 아키텍처 (MVP)
🏗️ URL → 데이터 → AI 생성 파이프라인
  • Step 1: 사용자가 스마트스토어 URL 입력
  • Step 2: Next.js API Route에서 서버사이드 크롤링 (cheerio로 HTML 파싱)
  • Step 3: 상품명, 가격, 카테고리, 옵션, 상세 이미지 URL, 리뷰 요약 추출
  • Step 4: 추출 데이터 + 네이버 SEO 키워드 → Claude API에 전달
  • Step 5: AI가 스마트스토어 HTML 형식으로 상세페이지 스트리밍 생성
  • Step 6: 결과를 Supabase에 저장 + 원클릭 복사 제공
⚠️ 법적·기술적 주의사항
  • 공개 데이터만 수집: 로그인 필요한 데이터는 절대 크롤링하지 않음
  • robots.txt 준수: 차단 경로는 크롤링하지 않음
  • 요청 빈도 제한: 1초당 1회 이하로 서버 부하 방지
  • 캐싱 전략: 동일 URL은 24시간 캐시 → 중복 크롤링 방지 + 비용 절감
